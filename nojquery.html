Discovery Loader Widget - No JQuery Example

This page shows what happens if JQuery is not loaded. The script degrades gracefully and shows a common search box. The Discovery Loader Widget requires JQuery for scoping functions. However, if it is accidently placed on a page without JQuery it will still work, creating a basic search box. Placeholder, Tagline, and Advanced search attributes may still be customized.

So if you don't have JQuery on your page, but you still want to place a plain unscoped search box, it'll work perfectly!
