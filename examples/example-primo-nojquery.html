Discovery Loader Widget (Primo) No JQuery Example Page

This page shows what happens if JQuery is not loaded. The script degrades gracefully and shows a common search box. The Discovery Loader Widget requires JQuery for scoping functions. However, if it is accidently placed on a page without JQuery it will still work, creating a basic search box.

So if you don't have JQuery on your page, but you still want to place a plain unscoped search box, it'll work perfectly!

The question is, can the script be re-written to not use jQuery? Yes! If you look at the code the main parts of jQuery (other than autosuggest) it uses is .attr and $(selectors). We might rewrite it but right now it serves our needs so it is as is.
